# project_contacts_node
A API of contacts

## Terminal Commands:
### global: 
`npm i -g typescript`
### develop:
`npm i -D prisma tsx`
### general:
`npm i express helmet dotenv @types/node @types/express`

### prisma db:
`npx prisma init` <br>
`npx prisma db push`

### Node_modules
`npm i`

### Run server:
`npm run start`
